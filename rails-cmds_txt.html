    4 - Run functional tests for only 1 controller¶ ↑

    ruby -I test test/functional/globes_controller_test.rb;

    (The '-I test' adds the test env into the ruby $LOAD_PATH)

    5 - Put <% debugger %> tags in ‘.erb’ code¶ ↑
